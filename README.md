# Campus Life Super App

## 📌 Project Description
The Campus Life Super App is a responsive web application designed for students at William Jewell College. The purpose of this project is to provide a centralized platform where students can easily access important campus information such as weather updates, upcoming events, and dining options.

This project demonstrates front-end web development skills using HTML, CSS, Bootstrap, and JavaScript, along with integration of an external API for dynamic content.

---

## 🚀 Features

- 📅 **Live Date Display**
  - Automatically shows the current date on every page

- 🌦️ **Weather Integration (API)**
  - Displays real-time weather data using the OpenWeather API

- 🎉 **Events Page**
  - Highlights upcoming campus events and athletics schedules

- 🍽️ **Dining Page**
  - Provides links to campus dining menus and nearby restaurants

- 📱 **Responsive Design**
  - Optimized for desktop, tablet, and mobile devices using Bootstrap and custom CSS

- 🎨 **Consistent UI Design**
  - Clean layout using Bootstrap components and custom styling

---

## 🛠️ Technologies Used

- HTML5 (Semantic structure)
- CSS3 (Custom styling + responsiveness)
- Bootstrap 5 (Layout and components)
- JavaScript (DOM manipulation + API calls)
- OpenWeather API (external data integration)
- Git & GitHub (version control and deployment)
- GitHub Pages (hosting)
